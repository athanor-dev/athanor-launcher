Linux 玩家不应该忍受 Wine 的兼容性层开销，也不应该让杂乱的游戏文件污染系统。
因此，我们正在构建 **Athanor** —— 一个像古代炼金炉一样精密、安全的容器。

### 核心特性

*   **原生 (Native):**
    使用 NW.js 本地运行时，拒绝“中间商赚差价”，彻底告别 Wine 的性能损耗。
*   **安全 (Safe):**
    基于 Flatpak 的严格沙箱隔离。你的系统保持纯净，游戏在笼中运行。
*   **进化 (Evolved):**（敬请期待）
    提供超越 Windows 原版的体验：智能存储去重、统一存档管理与版本控制。
