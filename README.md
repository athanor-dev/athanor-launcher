Linux players shouldn’t have to endure the overhead of Wine compatibility layers, nor should they allow cluttered game files to pollute their systems.

That is why we are building Athanor — a container as precise and secure as an ancient alchemical furnace.

### Core Features

*   **Native:**
    Powered by a local NW.js runtime, we cut out the "middleman." Say goodbye to Wine-induced performance loss once and for all.
*   **Safe:**
    Built on strict Flatpak sandbox isolation. Your system stays pristine while the game runs securely "in the cage."
*   **Evolved:** (coming soon)
    Delivering an experience that surpasses the original Windows version: intelligent storage deduplication, unified save management, and version control.
